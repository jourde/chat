Activity System Analyser based on cultural-historical activity theory (CHAT)

Click a node to edit it. Hover for guidance.

Summary

⚡ Primary Contradiction — a tension within a single component.
Click the same node twice to mark it.
Example: the Tools node — staff must use a digital platform they find unusable.
⚡ Secondary Contradiction — a tension between two components.
Click one node, then a different node in the same system.
Example: Rules demand standardised assessment, but the Object is personalised learning.
↔ Inter-systemic Contradiction — a tension between the two activity systems.
Click one node in System 1, then a node in System 2.
Example: System 1's rules clash with System 2's division of labour.
Note: Engeström (1987) also identifies tertiary contradictions — tensions between the dominant form of an activity and a culturally more advanced form of it — which are not modelled here.
